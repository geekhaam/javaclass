 (참고) WB에 속석에 값 넣는 두 가지 방법
    
	(1) 속성 창에 직접 입력
	    ==> 속성 창에 입력으로 처리한다.

	(2) 객체.속성 = 값
	    ==> 속성 창으로 대신할 수 있다.
	JTextField jt;		=> 글상자
	jt.setText("ondal");	=> setter
	jt.getText();		=> getter

 (참고) 개발시 변수명 약속 (Camel casing)
  버튼 : btn + Element name
  라벨 : lbl + Element name
